## React版豆瓣电影

#### React + ant-mobile 开发，小项目，主要做 React 的练习
#### 接口来自  [豆瓣api代理](https://github.com/zce/douban-api-proxy)

### 功能
+ 电影列表
+ 收藏电影
+ 电影评论
+ Top250
+ 电影详情
